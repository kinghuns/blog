Getting started

There are two ways to get started with this theme, depending on whether you use R or Hugo. R users If you use R, the easiest way to get started is to install the blogdown package, and create a new site with this theme: install.packages('blogdown') blogdown::new_site(theme = 'yihui/hugo-paged') Or you can create a new RStudio blogdown project (if you use RStudio) with the yihui/hugo-paged theme. Hugo users Create a new Hugo site with hugo new.
